# 📊 Painel de Gerenciamento de Projetos

## 🎯 Objetivo

Desenvolver um painel para acompanhamento de múltiplos projetos, permitindo uma visão executiva sobre progresso, custos e status das entregas.

---

## 🚨 Problema

Dificuldade em acompanhar o status geral dos projetos e consolidar informações como:

* andamento das entregas
* prazos
* custos
* progresso por projeto

---

## 💡 Solução

Criação de um dashboard visual para monitoramento de projetos, permitindo análise rápida e suporte à tomada de decisão.

---

## 🛠 Ferramentas utilizadas

* Power BI
* Excel

---
